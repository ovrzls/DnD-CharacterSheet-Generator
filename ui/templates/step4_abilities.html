How would you like to generate your ability scores?

Standard Array

Use a fixed set of six scores: 15, 14, 13, 12, 10, 8. Each score can only be used once. We have suggested the best arrangement for your class — you can change any assignment.

Point Buy

Start with 27 points. Spend them to raise ability scores from 8. Higher scores cost more points. No score can go below 8 or above 15.

Random Roll

Roll for your scores. Each score is the total of the three highest results from four six-sided dice. Scores cannot be below 8. You can reroll the full set as many times as you like before assigning.
